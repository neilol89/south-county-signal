# 15 — Social Media Integration Roadmap

## Goal
Drive newsletter signups directly from Instagram, TikTok, and Facebook. Every piece of social content should have a clear path to subscription.

---

## Platform Setup

### Instagram (@southcountysignal)
- **Bio:** "Your weekly cheat sheet for Southern RI 🌊 Events • Food • Music • Nightlife • Beach Life | Free newsletter 👇"
- **Link in bio:** southcountysignal.com (or Linktree/Beacons with multiple links)
- **Action button:** Use "Sign Up" button linked to southcountysignal.com/#subscribe
- **Highlights:** Create highlight reels — "This Week", "Best Of", "Deals", "About"

### TikTok (@southcountysignal)
- **Bio:** "What to do in South County RI this week 🌊 Free newsletter — link below"
- **Link in bio:** southcountysignal.com
- **Note:** TikTok only allows one link — use the main site URL

### Facebook (The South County Signal)
- **Page type:** Media/News Company
- **CTA button:** "Sign Up" → southcountysignal.com/#subscribe
- **About:** Full description with subscribe link
- **Pin a post:** "Welcome to the Signal" post with subscribe link

---

## Content Strategy → Signups

### Content Types That Drive Signups

| Content Type | Platform | Signup Hook |
|-------------|----------|-------------|
| **"This week in South County"** carousel | IG, FB | "Full list in the newsletter — link in bio" |
| **Hot take / poll** | IG Stories, TikTok | "We debate this every week in the Signal" |
| **Behind-the-scenes at a venue** | TikTok, IG Reels | "We featured this spot in this week's issue" |
| **Quick event roundup video** | TikTok, IG Reels | "Get the full lineup every week — free newsletter" |
| **Food/drink spotlight photo** | IG Feed | "Read the full review in this week's Signal" |
| **"Best of" listicle** | IG Carousel, FB | "We send picks like this every week — subscribe free" |
| **Deal teaser** | IG Stories | "Exclusive deals every week for subscribers" |

### Posting Cadence

| Platform | Frequency | Best Times |
|----------|-----------|------------|
| Instagram Feed | 3-4x/week | Tue-Thu 11am-1pm, Fri 5-7pm |
| Instagram Stories | Daily | Morning (8-9am), Evening (6-8pm) |
| TikTok | 3-5x/week | Wed-Fri 12pm, 6pm |
| Facebook | 2-3x/week | Tue-Thu 10am-12pm |

## Direct Signup Integration

### Link-in-Bio Strategy
Use Linktree or Beacons (free) with these links:
1. **Subscribe to the Signal** (primary) → southcountysignal.com/#subscribe
2. **Read the latest issue** → southcountysignal.com/#archive
3. **Advertise with us** → southcountysignal.com/#advertise
4. **Follow on Instagram** (cross-link from other platforms)

### Instagram Stories Signup Flow
1. Post a teaser Story ("5 things to do this weekend in South County")
2. Add a "Subscribe" link sticker → southcountysignal.com/#subscribe
3. Follow up Story: "Check your inbox every week for the full list"

### TikTok Signup Flow
1. Video: Quick-hit event roundup with text overlays
2. End screen: "Full lineup free in your inbox — link in bio"
3. Pin comment with subscribe CTA

### Facebook Signup Flow
1. Share newsletter highlight as a post
2. Include direct link to southcountysignal.com/#subscribe in post body
3. Facebook allows clickable links in posts (unlike IG)

---

## Automation Opportunities

### Phase 1 (Launch)
- Manual posting from phone/desktop
- Use Canva for graphics (consistent brand templates)
- Schedule posts with Later.com or Buffer (free tier)

### Phase 2 (Month 2-3)
- Create reusable Canva templates for weekly content types
- Set up Make.com or Zapier automation:
  - New Beehiiv post published → auto-post teaser to Facebook
  - New subscriber milestone → auto-post celebration to IG Stories
- Repurpose newsletter content into social posts automatically

### Phase 3 (Month 4+)
- Instagram Shopping / product tags for featured businesses (if applicable)
- TikTok Creator Fund (once following reaches threshold)
- Facebook Group: "South County Signal Community" for deeper engagement
- Consider paid social ads to drive signups ($5-10/day targeted to South County zip codes)

## Tracking & Attribution

### UTM Parameters
Tag all social links with UTMs to track which platform drives signups:
- Instagram: `?utm_source=instagram&utm_medium=social&utm_campaign=bio_link`
- TikTok: `?utm_source=tiktok&utm_medium=social&utm_campaign=bio_link`
- Facebook: `?utm_source=facebook&utm_medium=social&utm_campaign=post`

### Beehiiv Custom Field
Use the "Signup Source" custom field (already configured) to tag where subscribers came from. Can be set via UTM parameter capture on the site.

### Key Metrics to Track
- Signups per platform per week
- Social follower growth rate
- Click-through rate on link-in-bio
- Best-performing content types for signups
