# 11 — Weekly Ops Runbook

## The Weekly Rhythm

The newsletter goes out every **Thursday at 8:30 AM EST**. Everything works backward from that.

---

## Day-by-Day Production Schedule

### Monday: Source & Scout (2–3 hours)

**Morning (1 hour):**
- Open the content sourcing checklist (see 05-CONTENT-PLAYBOOK.md)
- Check all venue Instagram accounts for this week's events
- Check Facebook groups for community-posted events
- Scan event calendars: southcountyri.com, unitedtheatre.org, courthousearts.org, oceanchamber.org
- Check Eventbrite and Bandsintown for ticketed events
- Note anything from local media (RI Monthly, Westerly Sun, Patch)

**Afternoon (1 hour):**
- Open your Google Sheet editorial tracker
- Enter all candidate events (you'll have 15–25 raw picks)
- Flag the standout — this becomes "This Week's Move" candidate
- Note any restaurant news, new menus, or openings for Spotlight
- Check if any Providence events are worth the drive

**End of Monday:** You should have a raw list of 15–25 events and a sense of the week's story.

### Tuesday: Curate & Decide (1–2 hours)

**Tasks:**
- Cut the raw list down to 8–10 events (5–8 for Lineup + 1 Headliner)
- Ensure category diversity (at least 4 different categories)
- Ensure geographic spread (not all Westerly or all Narragansett)
- Pick the Spotlight restaurant/bar for this week
- Decide on Worth the Drive picks (or skip the section)
- Decide on Signal Noise format (poll, hot take, debate)
- Check ad inventory: are any slots sold this week? Confirm with advertisers.

**End of Tuesday:** You should have your final content selections locked.

### Wednesday: Write (2–3 hours)

**Morning (2 hours):**
- Write the Headliner description (3–4 sentences)
- Write all Lineup descriptions (1–2 sentences each)
- Write the Spotlight profile (3–4 sentences)
- Write Worth the Drive picks if applicable
- Write the Insider Deal copy
- Write the Signal Noise question/take
- Insert any sponsored content (write in your voice, get advertiser approval)

**Afternoon (1 hour):**
- Write the Opener LAST (you now know what's in the issue and can set the right tone)
- Write the subject line and preview text
- Assemble everything in Beehiiv's editor (or generate via the HTML pipeline)
- Add all links, double-check URLs
- Proofread once

**Evening:**
- Create 2–3 Instagram posts for Thursday–Saturday (Canva)
- Schedule posts in Later.com or Buffer
- Prepare Instagram Story slides for "This Weekend" quick hits

### Thursday: Send Day (1 hour)

**7:00 AM:** Final review in Beehiiv preview mode
- Read the full email on mobile (this is how most people read it)
- Click every link to verify they work
- Check that sponsor content is accurate
- Verify the subject line and preview text render correctly

**8:00 AM:** Send test email to yourself. Read it one more time.

**8:30 AM:** Schedule send (or hit send if you're a live-wire).

**9:00 AM:** Post to Instagram: "This week's issue is live" with top picks. Link in bio.

**Throughout the day:**
- Respond to every reply to the newsletter
- Engage with Instagram comments
- Share in Facebook groups if appropriate

### Friday: Review & Prep (30 minutes)

**Tasks:**
- Check Beehiiv analytics: open rate, click rate, click map
- Note the top-clicked link (this tells you what readers care about most)
- Log metrics in the analytics Google Sheet
- If any ad slots were sold, send the advertiser a performance report
- Post Instagram Story: "This weekend in South County" with 3–4 quick event picks

**End of Friday:** You're done for the week. Enjoy your weekend.

---

## Quality Assurance Checklist (Wednesday/Thursday)

Run through this before every send:

- [ ] Subject line is under 50 characters (mobile-friendly)
- [ ] Preview text complements but doesn't repeat the subject
- [ ] Opener is 2–3 sentences, no links
- [ ] Headliner has: name, date, time, venue, town, description, working link
- [ ] Each Lineup pick has: category tag, name, date/time, venue/town, description, working link
- [ ] Categories are diverse (4+ different categories)
- [ ] Geography is spread (not all one town)
- [ ] Spotlight has: name, town, description
- [ ] Worth the Drive: included only if genuinely compelling (otherwise removed)
- [ ] Insider Deal: has business name, offer, details, code
- [ ] Signal Noise: has a question/take + interactive element
- [ ] All sponsored content is labeled correctly (FTC compliance)
- [ ] All links work (click every single one)
- [ ] No remaining placeholder text (search for "[" brackets)
- [ ] Footer has correct social links and unsubscribe
- [ ] Mobile preview looks correct (Beehiiv preview tool)
- [ ] Test email sent and reviewed
